## TL;DR

A collection of useful web services for learning programming and development.

### [quicktype](https://quicktype.io)

A site that generates type definitions from sample JSON.

### [SVGR](https://react-svgr.com)

A site that generates React components from SVG files.

### [Graphviz Online](https://dreampuf.github.io/GraphvizOnline/)

A site that generates graphs from dot files.

### [GitHub Repository Card for every web site](https://gh-card.dev)

A site that creates nice-looking link cards for GitHub repositories. Cards like the following are displayed:

::gh-card[illumination-k/blog]

### [SQL Fiddle](http://sqlfiddle.com/)

A site where you can easily run SQL on the web. You can create tables and import data from CSV or JSON.

### [Material Icons](https://material-ui.com/components/material-icons/)

Search for the icons you need from `@material-icon`.
